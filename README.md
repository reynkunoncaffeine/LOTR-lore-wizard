# LOTR-lore-wizard
get your middle earth doubt cleared


## Setup
1. Clone the repo
2. Add your own `LOTR.pdf` to the project folder
3. Create a `.env` file with your Gemini API key:
   GEMINI_API_KEY=your-key-here
4. Install dependencies:
   pip install -r requirements.txt
5. Run `backend.ipynb` top to bottom to build the vector database
6. Launch the app:
   streamlit run simpleUI.py
   OR
   streamlit run lotrUI.py
